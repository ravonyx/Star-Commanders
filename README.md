# Star-Commanders

Unity Version 5.2.3

Les sources sont disponibles dans le répertoire Git suivant :
https://github.com/ravonyx/Star-Commanders


// --------------------------------------------------
Créer une partie :
	- Dans le menu principal, cliquez sur "Play".
	- Donnez un nom de salle, indiquez le nombre de joueurs et votre nom Ingame.
	- Cliquez sur "Create Room".
	
Arrivée dans la salle d’attente :
	- Lorsque tous les joueurs sont présents, cliquez sur "Play".
	Attention, la partie ne pourra être lancée que si le nombre de joueurs dans la salle est 
	égal au nombre de joueurs indiqué pendant la création de la salle. (5/5 Ok, 2/3 not Ok)

Rejoindre une partie :
	- Dans le menu principal, cliquez sur "Play".
	- Sélectionnez la salle que vous voulez rejoindre.
	- Attendre que l'hôte lance la partie.
// --------------------------------------------------

// --------------------------------------------------
Dans la partie :
Vous apparaissez à l'arrière du vaisseau.

Dans la colonne centrale du vaisseau, vous trouverez :
	-	Salle des générateurs avec une console pour la gestion de l’énergie.
	-	Salle de pilotage avec la console du pilote et 5 écrans affichant l’état des différents composants du vaisseau.
	-	Salle de la tourelle centrale 
Sur les côtés :
	-	Salles des réacteurs (liés aux propulseurs du vaisseau)
	-	Salles des tourelles de bordées
// --------------------------------------------------

// --------------------------------------------------
Contrôles :
-	Z/Q/S/D pour se déplacer, F pour interagir.
-	Dans la console pilote :
 	Pitch - Z / S
 	Yaw - Q / D
 	Roll - A / E
	Acceleration – Maj
 	Deceleration - Ctrl
 	Cacher Information – I
 	Sortir de la console – F
-	Dans les consoles tourelles :
 	Tir – Clique Droit
 	Tir Continu – Clique droit maintenu 
 	Cacher Information – I
 	Sortir de la console – F 
-	Dans la console de gestion d’énergie :
	Augmenter/Réduire puissance des armes – A / Z
	Augmenter/Réduire puissance des boucliers – Q / D
	Augmenter/Réduire puissance des propulseurs – W / X
	Cacher Information – I
 	Sortir de la console – F
// --------------------------------------------------
